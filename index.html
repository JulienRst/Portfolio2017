The Barber's Garden

school project - 2016
Developped with blade (laravel’s template engine)

The Barber’s Garden is a project realised as part of the web design class at IMAC School. This website is a web magazine design to share information on the interior design of Barber Shops across the world.

The website was created with a simple rooter of my own and the templating engine of Laravel: Blade, and data was stored on several JSON files.

One of the most important things about this project was the attention we put on making it as responsive as possible for most devices.

CONCIEVED AND DEVELOPED WITH
Lea Rozen-Sudry, Alice Jestin

IMAC Website

school project - 2016
Developped with Laravel

IMAC Website is a tutored project which was to give a new web identity to the school. We worked during six month, working on models, testing them with users, creating contents, etc… I was in charge of the back-office of the website, working with Laravel, managing versioning and deployment.

The project had to be maintainable by whoever wants to, so we made an administration panel, letting the next students in charge to have full control on the content of the website and not start from scratch every year.

CONCIEVED AND DEVELOPED WITH
Lea Rozen-Sudry, Alexander Feller, Charlotte Nortier
